# Transformer-Based Sentiment Analysis with Explainability

## 📌 Overview
This project implements a Transformer-based NLP model using BERT for sentiment analysis on the Amazon Polarity dataset. It also includes attention visualization and explainability using SHAP and LIME.

---

## 📊 Dataset
- Source: https://huggingface.co/datasets/fancyzhx/amazon_polarity
- Subset used:
  - Train: 5000 samples
  - Test: 1000 samples

---

## 🧠 Model
- Model: BERT (bert-base-uncased)
- Task: Binary Sentiment Classification
- Framework: Hugging Face Transformers
